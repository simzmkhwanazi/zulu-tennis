Return Strategy

The return is the second-most-important shot in tennis — and the most under-practised. Just getting more returns deep and in play wins more games than any flashy winner.

🎾 Courtside · Return Strategy A
Deep middle. Take away the +1.
✓ Do — 1st serve: block deep middle. 2nd serve: step in and attack.
✗ Don't — use the same big swing on every serve.
Split on the tossJust make them play
🎾 Drill: deep-return target — block first serves into the back third.

The return's only job

The return doesn't have to be a weapon. Its job is to start the point on neutral or better terms and deny the server a free +1. The first measurable goal: get a high percentage of returns deep and in.

🎯 Tactical purpose

Take away the server's cheap point. A deep return through the middle removes the server's angle for their +1 and forces them to construct. Level A

First-serve vs second-serve returns

Two completely different mindsets. Don't use the same return goal for both.

First-serve returnSecond-serve return
MindsetNeutralise — survive & resetAttack — step in, take time
TargetDeep, middle (kill the angle)Deep crosscourt or behind them
PositionDeeper, block/short swingStep inside the baseline
RiskLow — just get it deepModerate — controlled aggression

Returning specific servers

vs a Big Server Level B

  • Stand back and shorten the swing — block, don't swing. Use their pace.
  • Target deep middle — biggest margin, kills their +1 angle.
  • Win the second-serve points ruthlessly; that's where big servers are beatable.
  • On big points, just make the return — force one more ball.

vs a Pusher / Junkballer

  • Their serve is soft — step in and take time, but don't over-hit the first ball.
  • Return deep and to a corner, then look to approach on the next short ball.
  • Patience: don't try to win the point on the return; win it by applying steady forward pressure.

vs a Left-hander

  • The lefty slice serve swings into the ad-court backhand — expect it on big points and shade that way.
  • Practise returning the wide ad serve; protect the backhand or run around to a forehand when you can.
  • Remember the diagonal: returning crosscourt now feeds their forehand — consider deep middle instead.
⚠️ Common mistakes
  • Same aggressive swing on first and second serves — frees up errors on the big first serve.
  • Standing in the same spot every time; vary depth and shade by score and server tendency.
  • Going for a return winner on break point instead of making the server play.
🎾 Practice drill — "Deep return target"
  • Setup: server serves first & second; place a deep-middle target zone (back third).
  • Reps: 6 games — return only; +1 point for any return landing in the back third.
  • Constraint: first-serve returns must be blocked (short backswing) to count.
  • Coaching cue: "Split-step on their toss; eyes on contact; deep and in."
  • Success metric: ≥ 70% of returns in play; ≥ 40% landing deep.

Match application

Pick a return position per serve type before the game starts, default to deep-middle on first serves, and pre-decide your second-serve attack. On the receiver's side, your "+1" is the return — the Decision Engine takes over from the third ball.

✅ Success metrics
  • Return-in-play rate ↑ (especially on first serves).
  • Return points won ↑, particularly off second serves.
  • Break points earned per return game ↑ over a season.
